and retrives them on startup if they exist in localstorage, and mange app state through Provider.


create: clicking + button to create Time Entry
read: available entries will be read and displayed on homepage screen
update: clicking on the existing entry on hompeage will take you to edit entry page to update
delete: clicking on delete Icon delete the entry
tabs: 2 tabs- one tab for none sorted  list of entries and group tab for sort by project name
menu icon: open side panael with 2 options that will take you to manage Projects and Tasks pages.
